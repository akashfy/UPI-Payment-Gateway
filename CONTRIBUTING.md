# Contributing to UPIShield Gateway

First off, thanks for taking the time to contribute! 🎉

## How to Contribute

1. **Fork the repository**: Click the "Fork" button in the top right corner of the repo.
2. **Create a branch**: `git checkout -b feature/your-feature-name`
3. **Make your changes**: Write your code and make improvements.
4. **Run `gofmt`**: Ensure your code is formatted correctly by running `gofmt -w .`.
5. **Test locally**: Run the server with `go run .` to verify your changes work as expected.
6. **Open a Pull Request**: Submit your changes back to the main repository.

## Good First Issue Ideas

If you're looking for ways to contribute, here are some ideas:
- Improve documentation and setup guides.
- Add robust error handling.
- Improve the frontend design and responsiveness.
- Write unit tests for core functionalities.
- Implement additional webhook notifications.

Thanks again for contributing and helping make UPIShield Gateway better!
